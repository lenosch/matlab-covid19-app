# MATLAB COVID-19 Data Visualization App

An interactive MATLAB application developed using App Designer to process and visualize real-world COVID-19 data.

## 🚀 Features
- Interactive country and state selection
- Daily and cumulative case visualization
- Moving average smoothing for trend analysis
- Clean and user-friendly GUI

## 🛠️ Technologies Used
- MATLAB
- App Designer
- Data Processing

## 📊 Dataset
COVID-19 data provided by Johns Hopkins University.

## ▶️ How to Run
1. Open the `covid_app.mlapp` file in MATLAB
2. Ensure the dataset (`covid_data.mat`) is in the same folder
3. Run the application
